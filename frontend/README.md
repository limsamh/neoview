# NeoView Frontend

Angular application for querying and visualizing graph data from Wikidata and local .ttl files.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

## Running

Start the development server:

```bash
ng serve
```

Navigate to `http://localhost:4200`.

## Features

-   **Query Editor**: Write and execute SPARQL queries.
-   **Graph Visualization**: Visualize results as an interactive graph.
-   **Tabular Results**: View results in a table format.
-   **Saved Queries**: Save and load your favorite queries.
